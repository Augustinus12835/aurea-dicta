# Slide Visual Style Guide

## Overview
This guide defines the visual style for all generated educational video slides. The aesthetic is hand-drawn and educational, designed to feel approachable and engaging.

---

## Aesthetic: Hand-Drawn Educational

### Core Characteristics
- Clean, sketch-like illustrations
- Whiteboard/notebook feel
- Approachable, not corporate
- Simple and clear
- Educational, not flashy

### What To Avoid
- Stock photo aesthetics
- Corporate presentation style
- Overly polished/sterile look
- Cluttered layouts
- Distracting animations

---

## Color Palette

### Primary Colors
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Blue | #3B82F6 | Key concepts, headers |
| Secondary Orange | #F97316 | Highlights, emphasis |
| Tertiary Green | #22C55E | Positive outcomes, success |
| Text Dark | #1F2937 | Body text, labels |
| Background | #FAFAF9 | Warm white background |

### Extended Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Warning Red | #EF4444 | Negative outcomes, caution |
| Neutral Gray | #6B7280 | Secondary text, borders |
| Light Gray | #E5E7EB | Dividers, subtle elements |

### Color Usage Rules
1. Maximum 3 colors per slide (excluding text)
2. Use color consistently across all videos
3. Primary color for main concepts
4. Secondary color for highlights only
5. Background should always be warm white

---

## Typography

### Font Styles
- **Headers:** Hand-drawn style (clean, readable)
- **Body text:** Sans-serif (Arial, Helvetica, or similar)
- **Code/formulas:** Monospace (Consolas, Monaco)

### Font Sizes (for 1920x1080)
| Element | Size | Weight |
|---------|------|--------|
| Main title | 72-96px | Bold |
| Section header | 48-64px | Bold |
| Body text | 32-40px | Regular |
| Caption/label | 24-28px | Regular |

### Text Rules
1. Maximum 20 words of visible text per slide
2. No paragraphs - bullet points only
3. 3 bullet points maximum per slide
4. All text must be readable at 1080p

---

## Layout Principles

### Grid System
- Use a simple 3x3 or 4x3 grid
- Align elements to grid intersections
- Maintain consistent margins (80-120px from edges)

### Visual Hierarchy
1. **Primary element:** Center or left-center
2. **Supporting elements:** Right or bottom
3. **Labels/captions:** Near their subjects

### Spacing Rules
- Maximum 3 visual elements per slide
- Generous whitespace (30%+ of slide area)
- Left-to-right reading flow
- Key point at center or upper-third

### Balance
- Avoid placing all content on one side
- Use visual weight distribution
- Empty space is intentional, not missing

## Frame Types

### 1. Title Slide
**Purpose:** Introduce video topic
**Elements:**
- Large title text (center)
- Optional visual metaphor
- Course/series branding (subtle, corner)

### 2. Quote Slide
**Purpose:** Present memorable quotes
**Elements:**
- Large quote text (centered, italic)
- Attribution (smaller, right-aligned)
- Simple related icon (optional)

### 3. Concept Slide
**Purpose:** Explain single concept
**Elements:**
- Header (top or left)
- Illustration (center)
- 2-3 bullet points (if needed)

### 4. Comparison Slide
**Purpose:** Compare two concepts
**Elements:**
- Two-column layout
- Clear labels for each side
- Visual divider (subtle line or color)
- Consistent styling both sides

### 5. Example Slide
**Purpose:** Show real-world application
**Elements:**
- Scenario description (brief)
- Numbers/data (prominent)
- Visual representation
- Outcome/result

### 6. Process Slide
**Purpose:** Show sequential steps
**Elements:**
- Numbered steps (3-5 max)
- Arrows or flow indicators
- Icons for each step (optional)

### 7. Summary Slide
**Purpose:** Synthesize key takeaways
**Elements:**
- 3 key points (maximum)
- Simple icons or checkmarks
- Call to action (optional)

## Illustration Guidelines

### Style
- Line weight: 2-4px (consistent)
- Rounded corners preferred
- Simple shapes over complex
- Minimal detail, maximum clarity

### What To Include
- Simple diagrams
- Icons representing concepts
- Flow charts (simple)
- Comparison visuals
- Metaphorical illustrations

### What To Avoid
- Photos of real people
- Complex detailed drawings
- Logos or branded content
- Copyrighted imagery
- Clipart aesthetic

---

## Data Visualization

### Charts and Graphs
- Use simple, clean charts
- Maximum 5 data points
- Clear labels on all axes
- Use color palette consistently
- Include source attribution (small)

### Chart Types
| Data Type | Recommended Chart |
|-----------|------------------|
| Comparison | Bar chart |
| Trend over time | Line chart |
| Proportions | Pie chart (max 4 slices) |
| Distribution | Simple histogram |
| Relationship | Scatter plot |

### Data Visualization Rules
1. Remove chart junk (unnecessary gridlines, etc.)
2. Highlight the key insight
3. Use color to draw attention
4. Include brief interpretation

## Technical Specifications

### Resolution
- **Output:** 1920 x 1080 pixels (16:9)
- **DPI:** 150 (for PDF extraction)
- **Aspect ratio:** 16:9 (always)

### File Formats
- **Slides:** PDF (for extraction)
- **Frames:** PNG (for video compilation)
- **Diagrams:** PNG or SVG

### Export Settings
- High quality (no compression artifacts)
- Consistent naming: `frame_0.png`, `frame_1.png`
- No padding in filenames
